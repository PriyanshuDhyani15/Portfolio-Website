What I've Built

Projects

MCA Final Year Project

Diksha — Multilingual College Chatbot

An AI-powered chatbot built for G.B. Pant Institute, designed to answer college-related queries in English, Hindi, Garhwali, and Kumauni — making institutional knowledge accessible to students in their native languages.

  • RAG (Retrieval-Augmented Generation) pipeline using Qdrant vector search + BM25 hybrid retrieval
  • LLaMA 3.3 70B via Groq API for fast, accurate language generation
  • FastAPI backend with clean REST endpoints; React.js frontend for fluid chat UI
  • Multilingual NLP to understand student queries across four languages
  • Dynamic response system backed by SQL database of FAQs, courses, and campus info
